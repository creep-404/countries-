# Countries Dataset Analysis

A basic exploratory data analysis (EDA) project using **Python**, **Pandas**, and **NumPy** to inspect and understand a structured countries dataset.

## Overview

This project covers fundamental data exploration techniques commonly used as the first step in data analysis and machine learning workflows.

## Operations Performed

- Load CSV data into a Pandas DataFrame
- Inspect dataset structure
- Display first and last records
- Retrieve dataset dimensions
- List column names
- Analyze data types and non-null values
- Generate descriptive statistics
- Detect missing values
- Count unique values
- Display random samples
- Analyze continent-wise record distribution using `value_counts()`

## Tech Stack

- Python 3
- Pandas
- NumPy

## Dataset

- `countries.csv`

## Purpose

This project demonstrates the use of core Pandas functions for dataset inspection and exploratory data analysis (EDA), providing a foundation for data preprocessing and machine learning pipelines.
